# Wheat GEA Pipeline

Reproducible pipeline for genome-environment association analysis and adaptive germplasm prioritization in bread wheat.

## Project structure

RawData/     Input files. Not tracked by Git.
Data/        Intermediate files generated by the pipeline. Not tracked by Git.
Output/      Analysis results and figures. Not tracked by Git.
Logs/        Execution logs. Not tracked by Git.
Functions/   R functions used by the pipeline.
